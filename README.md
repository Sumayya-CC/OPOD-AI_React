# OPOD-AI — Profanity and Obscenity Detector (UI Demo)

A React frontend demo built to showcase profanity detection in text — 
created as a sample UI during my time at Tarento Technologies, where 
my mentor was contributing to the iGOT platform (a live Indian 
Government project).

## What It Does

- Paste or upload text for profanity analysis
- Highlights offensive words/phrases inline
- Shows overall profanity score and word-level occurrence counts
- Classifies severity: Lightly Offensive / Offensive / Highly Offensive
- Categories: Unparlimentary, Culturally Sensitive, Politically Sensitive
- Select text and add it to training data
- Paginated results with Export Summary
